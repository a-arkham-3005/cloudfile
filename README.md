Cloudfile
=========

Описание
--------

Cloudfile - приложение облачного хранилища с открытым исходным кодом для Android.

Требования
----------

*   HTTP-сервер с PHP и MySQL (к примеру, XAMPP) (для развертывания)
*   FTP-сервер (например, FileZilla Server) (для развертывания)
*   Устройство на Android или эмулятор
*   Android Studio Narwhal 2025.1.4 или Hedgehog (для сборки) - на остальных версиях работа не тестировалась

Установка
---------

Скачайте apk-файл в разделе Releases, установите и пользуйтесь.

Развёртывание и сборка
----------------------

1.  Скопируйте содержимое папки backend (папку cf) в корень файлов веб-сервера (в случае с XAMPP - папка htdocs) и измените в файлах в ней адреса базы данных на свои
2.  Импортируйте файл cloudfile.sql в базу данных с помощью phpMyAdmin
3.  Настройте FTP-сервер - добавьте пользователя admin с паролем rj45kpj и назначьте ему корневую папку по выбору
4.  Импортируйте проект в Android Studio
5.  Измените адреса в файлах FtpHandler.java и HttpHandler.java на свои
6.  Соберите проект и запустите на устройстве/эмуляторе
7.  Стандартные логин/пароль: admin/1234
